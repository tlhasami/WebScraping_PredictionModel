# WebScraping_PredictionModel ‚úÖ

**Overview**

This repository contains two data science projects demonstrating web-scraping, data cleaning, exploratory data analysis, and machine learning model building.

- **House price prediction** (HousePrediction): data scraped from Zameen.com and a model that predicts house prices from features like area, location, number of rooms, and bathrooms.
- **PUBG player statistics** (Models/PubPlayerStatistics): exploratory and statistical analysis of PUBG player behavior and performance.

## How to run ‚ñ∂Ô∏è

1. Create a Python virtual environment and activate it:

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install dependencies (example):

```bash
python -m pip install --upgrade pip
pip install jupyter pandas numpy scikit-learn matplotlib seaborn
```

3. Launch Jupyter and open the notebooks:

```bash
jupyter lab  # or 'jupyter notebook'
```

4. Follow the steps inside each notebook. The house price notebook shows preprocessing, feature engineering, model training (e.g., gradient-based methods), and evaluation.
